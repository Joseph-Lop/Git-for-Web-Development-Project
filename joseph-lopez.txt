    1. What is Semantic HTML? 
		Elements that give specific descriptions of whats being defined, example being
	<header>, <footer>, <section>, etc.

    7. What is an id? 
		Is an attribute that gives a name/identification to an element that is exclusive to that element, can not 
	be used on other elements.

    8. What elements can I add an id to? 
		You can use id on all/any element.

    9. How many times can I use the same id on a page? 
		You can only use it once because giving an element an id attribute makes that id exclusive to that element.

    10. What is a class? 
		class is used to give an element a name/identification but it can be used repeatedly(more than once) and
	isn't unique to elemenets given the class attribute.

    11. What elements can I add a class to?
		You can add class to any/all elements.
 
    12. How many times can I use the same class on a page?
